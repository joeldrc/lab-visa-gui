# VNA tester

A basic program for connecting to an VNA and and interfacing with a HP4195A Network/Spectrum Analyser to view and save the data.

## Requirements

### Hardware
- All VNA (Tested on R&S)

### Software
- Python 3.0 +

### Supporting Documents


### Installing

To run the program...

```
python main.pyw
```
